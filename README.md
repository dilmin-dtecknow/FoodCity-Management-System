# FoodCity Management System

## Overview

FoodCity Management System is a Java-based desktop application designed to manage inventory, orders, and basic store operations.

## Features

* Inventory management
* Order processing
* Customer management
* Database integration

## Technologies Used

* Java
* MySQL
* JDBC

## How to Run

1. Clone the repository.
2. Open the project in IntelliJ or Eclipse.
3. Configure the MySQL database connection.
4. Run the main application class.
